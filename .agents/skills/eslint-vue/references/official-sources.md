# ESLint Vue Official Sources

Use these sources for Vue SFC linting and Vue + TypeScript ESLint integration.

- eslint-plugin-vue User Guide: https://eslint.vuejs.org/user-guide/
- eslint-plugin-vue Rules: https://eslint.vuejs.org/rules/
- Vue ESLint Config TypeScript: https://github.com/vuejs/eslint-config-typescript
- ESLint Configuration Files: https://eslint.org/docs/latest/use/configure/configuration-files
- Prettier Install: https://prettier.io/docs/install

Project lint commands, config paths, dependency policy, and temporary artifact policy come from `CODEX_PROJECT.md` and the target repository.

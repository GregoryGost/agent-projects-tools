# Vue Router Official Sources

Use these sources for route behavior, router APIs, navigation guards, and route records.

- Vue Router Docs: https://router.vuejs.org/
- Vue Router Guide: https://router.vuejs.org/guide/
- Vue Router API Reference: https://router.vuejs.org/api/
- Vue Router Navigation Guards: https://router.vuejs.org/guide/advanced/navigation-guards.html

Prefer official Vue Router documentation for router behavior. Project route names, meta fields, and validation paths still come from `CODEX_PROJECT.md` and the target repository.

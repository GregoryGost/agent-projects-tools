# ESLint TypeScript Official Sources

Use these sources for ESLint and TypeScript ESLint configuration.

- ESLint Getting Started: https://eslint.org/docs/latest/use/getting-started
- ESLint Configuration Files: https://eslint.org/docs/latest/use/configure/configuration-files
- ESLint CLI Reference: https://eslint.org/docs/latest/use/command-line-interface
- typescript-eslint Getting Started: https://typescript-eslint.io/getting-started/
- typescript-eslint Typed Linting: https://typescript-eslint.io/getting-started/typed-linting/
- typescript-eslint Shared Configs: https://typescript-eslint.io/users/configs/
- Prettier Install: https://prettier.io/docs/install

Project lint commands, config paths, dependency policy, and temporary artifact policy come from `CODEX_PROJECT.md` and the target repository.

# TypeScript Core Official Sources

Use these sources when a TypeScript rule, review comment, or implementation choice needs verification.

- TypeScript Docs: https://www.typescriptlang.org/docs/
- TypeScript Handbook: https://www.typescriptlang.org/docs/handbook/intro.html
- TypeScript TSConfig Reference: https://www.typescriptlang.org/tsconfig/
- TypeScript Utility Types: https://www.typescriptlang.org/docs/handbook/utility-types.html

Prefer the official TypeScript documentation over blog posts for language behavior, compiler options, and type-system details.

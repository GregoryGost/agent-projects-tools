# Pinia Official Sources

Use these sources for Pinia store behavior, API usage, and Vue integration.

- Pinia Docs: https://pinia.vuejs.org/
- Pinia Core Concepts: https://pinia.vuejs.org/core-concepts/
- Pinia API: https://pinia.vuejs.org/api/
- Pinia Plugins: https://pinia.vuejs.org/core-concepts/plugins.html

Prefer official Pinia documentation for store APIs. Project store boundaries, persistence, and validation commands come from `CODEX_PROJECT.md` and the target repository.

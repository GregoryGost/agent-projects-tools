# SCSS Tailwind Official Sources

Use these sources for Tailwind CSS and Sass/SCSS behavior.

- Tailwind CSS Docs: https://tailwindcss.com/docs
- Tailwind CSS With Vite: https://tailwindcss.com/docs/installation/using-vite
- Tailwind Responsive Design: https://tailwindcss.com/docs/responsive-design
- Tailwind Dark Mode: https://tailwindcss.com/docs/dark-mode
- Sass Documentation: https://sass-lang.com/documentation/
- Sass CSS Compatibility: https://sass-lang.com/documentation/breaking-changes/

Prefer official Tailwind and Sass docs for syntax and version-specific behavior. Project theme tokens, config files, and validation commands come from `CODEX_PROJECT.md` and the target repository.

# Codex Project Profile Template

This file is a template. Copy it to the repository root as `CODEX_PROJECT.md` and replace placeholders.

This template is not a rule. Keep it outside `.codex/rules/`.

## Activation Semantics

The final `CODEX_PROJECT.md` is project-specific configuration, not a catalog of all available rules and skills.

Only entries explicitly listed in `CODEX_PROJECT.md` are active: stack profiles, rule files, skills, commands, and external systems.

Omitted entries and entries set to `none` are not active. Delete unused optional sections from the final file.

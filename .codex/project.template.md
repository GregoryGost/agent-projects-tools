# Codex Project Profile Template

This file is a template. Copy it to the repository root as `CODEX_PROJECT.md` and replace all placeholders.

This template is not a rule. Keep it outside `.codex/rules/` so agents do not load it as an active rule file.

Selection rule: the final `CODEX_PROJECT.md` must keep only project-specific active profiles, rules, skills, commands, and integrations. Omitted entries are inactive.

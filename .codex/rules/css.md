# CSS rules

Apply this rule when `CODEX_PROJECT.md` declares CSS active or when the task changes CSS.

## Required skills

- `css-expert`.

## Source of truth

1. Read `CODEX_PROJECT.md`.
2. Check CSS entry points, browser support policy, validation commands, and affected files.
3. Load `css-expert/references/patterns-and-review.md` for review checks.

## Constraints

- Prefer existing tokens and classes before adding new ones.
- Keep selectors maintainable.
- Check browser support before adding modern CSS features.
